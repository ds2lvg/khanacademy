- 2(3x+5) = 6x+10
- -3y + 4xy - 2x² + 2x + y² - 4xy + 2y + 3x²
  - -y + x² + 2x + y²
- -5.55 - 8.55c + 4.35c = -5.55 - 4.2c

## 번분수(Fraction division as Algebraic expressions)
- 분수는 결국 나눗셈
- 분수의 나눗셈은 나누는 수의 역수의 곱
- (1/a - 1/b) / c ÷ 1/d
  - (b/ab - a/ab) / c ÷ 1/d
  - (b-a)/ab · 1/c · d
  - (db-da)/abc

```
3x/y
----
2x/7
```
- (3x/y) / (2x/7)
- 3x/y · 7/2x
- 21x/2xy
- 21/2y

```
1-(s/t)
-------
1+(s/t)
```
- 1=t/t
- (t-s)/t / (t+s)/t
- (t-s)/t · t/(t+s)
- (t-s) / (t+s)

```
3+(3/x)
-------
4+(4/x)
```
- (3x+3)/x + x/(4x+4)
- 3(x+1) / 4(x+1)
- 3/4
